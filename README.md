# Power-Bi-Ecommerce-Sales-Dashboard # 🛒 Ecommerce Sales Dashboard | Power BI

## 📌 Project Overview
This project is an interactive **Ecommerce Sales Dashboard** built using **Microsoft Power BI**
for Madhav Store — an online retail business operating across India.
The dashboard helps the store owner make data-driven decisions by tracking and analyzing online sales performance.
---

## 🎯 Objective
To help the owner of Madhav Store track and analyze their **online sales data across India**
through an interactive and visually appealing Power BI dashboard.

---

## 🛠️ Steps Followed

- 📥 **Data Extraction** — Loaded raw sales data from CSV files (Orders & Details)
- 🔧 **Data Transformation** — Cleaned and transformed data using Power Query
- 📊 **Dashboard Creation** — Built interactive visualizations with filters and slicers
- 🔗 **Data Modeling** — Created table connections and joined new tables
- 🧮 **DAX Calculations** — Performed calculations to manipulate data and enhance visuals

---

## 📊 Dashboard Features

- 💰 KPIs — Total Sales, Total Profit, Total Quantity, Average Order Value
- 🗺️ Sales by State (India Map View)
- 📦 Sales by Category & Sub-Category
- 👥 Top Customers by Revenue
- 💳 Payment Mode Analysis (COD, UPI, Cards, EMI)
- 📅 Monthly Profit & Sales Trends
- 🔍 Dynamic Slicers — Filter by Quarter & State

---

## 📈 Visualizations Used

Bar Chart · Donut Chart · Clustered Bar Chart ·
Pie Chart · Line Chart · Area Chart · Map · Slicers

---

## 🛠️ Tools & Technologies

- **Microsoft Power BI Desktop**
- **Power Query** – Data Cleaning & Transformation
- **DAX** – Calculated Measures & KPIs
- **CSV Data Source** – Orders.csv & Details.csv

## 🚀 How to Use

1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/)
2. Clone this repository or download the `.pbix` file
3. Open the file in Power BI Desktop
4. Use slicers to explore sales by quarter and state!
